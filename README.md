A simple, responsive **User Registration** form implemented with plain HTML, CSS and JavaScript. This is a frontend-only prototype (no backend) demonstrating accessible form layout, validation, and UX patterns for signup screens.

## Features
- Responsive sign-up form (first name, last name, email, password + confirm, DOB, gender, T&Cs)  
- Client-side validation: required fields + password match  
- Accessible labels and keyboard-friendly layout  
- Styles using modern CSS (flexbox) and hover/active interactions

## Demo / Run locally
1. Clone the repo:
```bash
git clone https://github.com/venkata3204sai/voc-minor-project.git
cd voc-minor-project
```
2. Open index.html in your browser (or serve with a static server):
```bash
python -m http.server 8000
# open http://localhost:8000
```

Note: This is a frontend prototype. To persist users across devices or add sessions, add a backend API (Node/Express, Flask, Firebase, etc.).
